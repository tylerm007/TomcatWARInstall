Derive an attribute value using other attributes in that table, or parent table (changes are propagated). For conditional assignment, use JavaScript if/else, or the ternary operator.

Parent references are accessible through the relationship name, for instance a lineitem child related to a parent described as itemOrder, may be used via row.itemOrder
