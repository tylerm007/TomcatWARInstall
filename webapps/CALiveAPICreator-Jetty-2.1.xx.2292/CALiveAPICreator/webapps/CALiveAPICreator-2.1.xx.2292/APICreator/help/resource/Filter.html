Filter

This is a way to specify additional filtering on the resource, e.g. if you want to restrict the rows that should be returned by the resource.

For SQL resources, this should simply be a fragment of a where expression, e.g.: amount_total < 1000 and paid = 'Y'

For Mongo resources, you should use the Mongo syntax, e.g.: {"$and": [{"$lt": {"amount_total": 1000}}, {"paid": "Y"}]}
