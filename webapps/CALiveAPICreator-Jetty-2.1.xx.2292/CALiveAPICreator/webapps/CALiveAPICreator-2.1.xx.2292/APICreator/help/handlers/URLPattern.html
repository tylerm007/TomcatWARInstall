URL Pattern

This is a regular expression that will be matched with the incoming request's URL. If they match, this handler will be called.

This regular expression follows the conventions of Java regular expressions. A full reference is available on Oracle's web site.

Note that, if more than one handler matches a request's URL, the first one found will be invoked. You should be careful that your URL patterns not overlap, otherwise there is no way to predict which handler will be invoked.
