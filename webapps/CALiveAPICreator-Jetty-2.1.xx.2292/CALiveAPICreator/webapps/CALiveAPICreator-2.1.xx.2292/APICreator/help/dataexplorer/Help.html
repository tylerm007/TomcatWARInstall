Data Explorer

What is this?
The Data Explorer gives you complete read/write access to your data:

  • No coding required - it is created from your schema, and stays in sync with schema changes

  • It uses the RESTful API, so enforces logic and security

  • It is configurable as described below

What is the intended usage?
It is intended for 'backoffice' data maintenance apps, a complement to your custom apps. Both Data Explorer and your custom apps share the common Live API, including Logic and Security enforcement.

Can I configure it?
Use Author Mode to customize many aspects of appearance:

  • What fields are displayed, in what order

  • Presentation rules, such as when to hide, disable or highlight data

  • Style sheets, to control colors, fonts and logos

How can I explore the interaction with Live API and Live Logic?
Zoom into an Order, and alter an Order Detail Quantity. Observe the change to the Customer Balance.
