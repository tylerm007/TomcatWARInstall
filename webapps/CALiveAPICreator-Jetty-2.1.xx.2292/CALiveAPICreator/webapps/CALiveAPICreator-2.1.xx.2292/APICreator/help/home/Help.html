{{productFullName}} simplifies and accelerates creating RESTful servers, with declarative definition of the services typically found in an App Server:

  1. API: point-and-click document-oriented endpoints, complementing the default API created from your schema, with pagination, optimistic locking, and generated primary key propagation to child data.

  2. Integration: resources can combine data from multiple sources (SQL, Mongo and RESTful), and coordinate updates between them.

  3. Logic Enforcement: database integrity with a combination of spreadsheet-like Rules and server-side JavaScript.

  4. Security Enforcement: controls end-point access, and row/column security.

Specify your settings in the {{productSlimName}}; activation is instantaneous - no code gen, no deployment.

Use the Data Explorer, created from the schema, to test your API, and for back-office database maintenance.

Flexible Deployment - both the RESTful service and database can be in the cloud, or on-premise (VMware image). You can also use the war file, to deploy into your own container.

Use your own existing database, or the pre-supplied empty database in the Your Database project.
