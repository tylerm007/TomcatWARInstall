Manage APIs

Use this API List to manage APIs:

  • Create a New API by connecting to a database, or

  • Import an exported API, or

  • Select an API by clicking a panel to specify its behavior (summarized below)

APIs are live - instantly executable - per your behavior specifications, including:

  • Your Data Source location(s)

  • Your Resource (aka End Point) definitions

  • Your Logic and Security rules and JavaScript

Once you select an API, you can not only control its behavior, but also run it to verify proper operation.
