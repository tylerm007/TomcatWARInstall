Entity/column

Specify the table/column whose value is the sum of a child entity/column.
A child is an entity containing a Foreign Key Relationship to a parent entity.

This value will be automatically adjusted whenever changes are detected in:

  • Child entity/column value
  • Child Qualification
  • Child Foreign Key
  • Child inserts, deletes

Rules are not directly invoked - they fire automatically, providing all necessary SQL handling.

Processing is optimized to reduce SQL.

A sum is very similar to a spreadsheet cell formula. See here for more information.
