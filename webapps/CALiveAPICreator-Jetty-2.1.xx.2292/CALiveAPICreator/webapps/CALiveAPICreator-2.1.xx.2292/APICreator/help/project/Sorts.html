User sorts

These are user defined syntax with specified (exposed) column attribute names that limit how sorts are performed.
Named Sorts
.../v1/[Resource]?order=balance%20desc
.../v1/[Resource]?sysorder=(expression[,expression])
.../v1/[Resource]?userorder=myNamedFilter([expression])


expression = (columnName:modifier)
modifier is asc, desc, asc_uc, desc_uc, null_first, null_last

are useful, but they can be a security problem. User sorts allow you to pre-define arbitrarily complex sorts in a secure way. See the documentation for additional details.
