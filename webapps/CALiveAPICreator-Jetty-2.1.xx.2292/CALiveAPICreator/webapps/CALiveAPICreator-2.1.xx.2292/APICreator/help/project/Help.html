API Properties

An API is an executable set of Resources (End Points) accessed via a RESTful API, consisting of:

  • Default Resources are the tables, views and stored procedures of one of more underlying Data Sources.
  • Custom Resources nest data from multiple Data Sources, including control of attributes and their names
In addition to the API shape, you can configure logic and security behavior, using Reactive (spreadsheet-like) Rules and JavaScript.

Your API is live - instantly executable, no compile or deploy. Run it in the REST Lab or the Data Explorer.
