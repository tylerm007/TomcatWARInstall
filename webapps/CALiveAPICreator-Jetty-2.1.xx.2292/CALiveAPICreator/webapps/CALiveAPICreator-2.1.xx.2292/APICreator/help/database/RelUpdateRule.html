Relationship - On Update

This option determines what happens when a parent row's key gets updated. The options are:
  • Update: the children rows' foreign keys will be updated to reflect the new parent key
  • Null: the children rows' foreign keys will be set to null (note that this will result in an error if the foreign key is not nullable)
  • Restrict: the update of the parent row's key will fail if it has any children rows.
