Database URL

This is the URL that points to your database. It is heavily database-dependent, so you'll probably want to consult the relevant documentation if you don't already know this.

Database type URL template Reference docs
MySQL jdbc:mysql://<server-name>[:port-number (default 3306)]/[database-name] MySQL site
Oracle jdbc:oracle:thin:@<host>:<port>:<sid>
or
jdbc:oracle:thin:@//<host>:<port>/<service>
or
jdbc:oracle:thin:@<TNSName> - see Oracle for details		 Oracle site
Microsoft SQL Server jdbc:sqlserver://<server-name>[instance-name][:port-number]][;property=value[;property=value]] Microsoft site
Postgres jdbc:postgresql://<server-name>[:port-number (default 5432)]/<database-name> Postgres site
NuoDB jdbc:com.nuodb://<server-name>[:port-number (default 48004)]/<database-name> NuoDB site
HP Vertica jdbc:vertica://<server-name>[:port-number (default 5433)]/<database-name> HP Vertica site

  • The server-name is the address of your database server. It can be either an IP address like 12.34.56.78, or (more likely) a name that ends with .com, .net. or something like that. This server must of course be accessible from CA Technologies Live API server in the cloud.
  • The port-number can typically be left unspecified, unless your database server uses a non-standard port.
