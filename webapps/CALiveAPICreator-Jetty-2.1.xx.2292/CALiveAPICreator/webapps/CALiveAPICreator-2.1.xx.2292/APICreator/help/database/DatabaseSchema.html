Database Schema

This is a name that points to your schema on the database server. The meaning of this is is database-dependent and should be:
  • MySQL: not used - leave blank (may not be visible)
  • Oracle: the schema for this database to use, sometimes referred to as the owner
  • SQL Server: often 'dbo', the default schema or owner to use
  • NuoDB: the database schema to use
  • PostgreSQL: the schema schema to use
  • Pervasive PSQL: not used - leave blank (may not be visible)
  • Composite: the database schema to use
  • HP Vertica: the database schema to use
