Database Catalog

This is a name that points to your database on the server. This meaning is database-dependent and should be:
  • MySQL: the name of your database.
  • Oracle: leave blank (may not be visible)
  • SQL Server: the name of your database
  • NuoDB: the name of your database
  • PostgreSQL: the name of your database
  • Pervasive PSQL: the name of your database
  • Composite: the name of your database
  • HP Vertica: the name of your database
