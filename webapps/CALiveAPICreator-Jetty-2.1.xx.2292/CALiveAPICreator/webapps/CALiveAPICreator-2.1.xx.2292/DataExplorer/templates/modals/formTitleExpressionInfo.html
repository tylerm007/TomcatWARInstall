Expressions

Premise

This expression will evaluate every time the form window is refreshed, updating the form title accordingly.

Syntax

Example: table.name + ' -> ' + row.credit_balance , where credit_balance is a column in this table and table.name is refers to the @tables.name request object

Expressions are evaluated as JavaScript, and when passing text into exposed methods, wrap the value in quotes:

"Expressed String -> " + row.credit_balance

Exposed Parameters

row : a reference to the values of this row

table : the Logic Designer API @tables request object
